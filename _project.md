# Magic Swatches (swatch-gen-plugin)

## What It Is
A Figma plugin that extracts the 5 most dominant colors from a selected image frame and generates a visual swatch strip.

## Stack
Figma Plugin API (manifest v2), TypeScript, vanilla HTML/CSS/JS, k-means clustering

## Goal
Working Figma plugin that samples an image, runs color quantization, and outputs a 740×100 swatch frame with hex labels.
